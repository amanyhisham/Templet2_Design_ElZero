
# 👻 Kasper — Design Agency Template

A fully responsive **multi-section design agency website template** built with **HTML** and **CSS**, based on the Kasper design from the Elzero Web School course.

---

## 🚀 Live Demo

🔗 [View Live](https://amanyhisham.github.io/Templet2_Design_ElZero/)

---

## ✨ Features

- 🖼️ Full-screen image slider with navigation arrows and dots
- 🧭 Navigation bar with: Home, Services, Portfolio, About, Pricing, Contact + Search icon
- 📱 Fully responsive design
- 🎨 Teal / dark blue color scheme
- 🔤 Custom web fonts

---

## 🛠️ Built With

- HTML5
- CSS3
